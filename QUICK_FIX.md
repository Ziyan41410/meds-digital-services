# 🚀 حل سريع للمشاكل

## المشاكل التي تواجهها:
1. ❌ **خطأ 500** عند محاولة إنشاء حساب
2. ❌ **CORS Error** - Origin not allowed

---

## ✅ الحل الفوري (5 دقائق):

### الخطوة 1: تحديث متغيرات البيئة
```bash
# في ملف .env.local تأكد من وجود:
CLIENT_URL=http://localhost:3000,http://localhost:3001,http://localhost:5500
```
✅ تم التحديث بالفعل!

---

### الخطوة 2: تأكد من قاعدة البيانات
```bash
# افتح PowerShell وشغل:

# 1. تأكد من أن MySQL يعمل
mysql -u root

# إذا اتصلت بنجاح اضغط CTRL+C للخروج

# 2. شغل schema:
mysql -u root < database\meds_schema.sql
```

---

### الخطوة 3: أعد تشغيل الخادم
```bash
# في terminal:
npm install
npm run dev
```

---

## 🔍 اختبار سريع

افتح browser على:
```
http://localhost:3000/health
```

يجب أن تري:
```json
{
  "status": "OK",
  "service": "MEDS Digital Services API",
  "timestamp": "2024-01-01T12:00:00.000Z"
}
```

---

## 🔧 إذا استمرت المشاكل:

### خطأ في قاعدة البيانات؟
```bash
# شغل أداة التشخيص:
node diagnostic.js
```

### CORS لا يزال غير مسموح؟
```bash
# تحقق من أن browser يستخدم http://localhost:3000 بالضبط
# وليس http://127.0.0.1:3000 أو http://localhost:5500
```

### خطأ 500 يستمر؟
```bash
# انظر إلى console الخادم عند محاولة التسجيل
# يجب أن ترى رسالة خطأ مفصلة تساعدك على حل المشكلة
```

---

## ✨ تم الإصلاح!

بعد اتباع الخطوات أعلاه:
1. ❌ خطأ CORS يجب أن يختفي
2. ❌ خطأ 500 يجب أن يختفي (إذا كانت قاعدة البيانات موجودة)

**يمكنك الآن التسجيل بنجاح! 🎉**
