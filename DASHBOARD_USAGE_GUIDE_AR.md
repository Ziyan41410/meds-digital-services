- **الوظيفة**: عرض عدد المشاريع قيد التنفيذ
- **المصدر**: قاعدة البيانات
- **التحديث**: تلقائي كل دقيقة
- **الحالات المحسوبة**: in_progress

### 3. **مدفوعات معلقة** 💰
- **الوظيفة**: عرض عدد الفواتير غير المدفوعة
- **المصدر**: قاعدة البيانات
- **التحديث**: تلقائي كل دقيقة
- **الحالات المحسوبة**: completed + paid=0

### 4. **نسبة الإنجاز** 📈
- **الوظيفة**: عرض نسبة المشاريع المكتملة هذا الشهر
- **الصيغة**: (completed / total) × 100
- **المصدر**: قاعدة البيانات
- **التحديث**: تلقائي كل دقيقة

### 5. **إجمالي الإنفاق** 💵
- **الوظيفة**: عرض المبلغ الإجمالي المنفق
- **الحساب**: مجموع جميع الميزانيات
- **المصدر**: قاعدة البيانات
- **التحديث**: تلقائي كل دقيقة

### 6. **المشاريع الأخيرة** 📌
- **الوظيفة**: عرض آخر 4 مشاريع للعميل
- **البيانات**: العنوان، الوصف، الحالة، التقدم، نوع الخدمة
- **الترتيب**: حسب آخر تحديث
- **التحديث**: تلقائي كل دقيقة

## 🎯 الأزرار السريعة (3 أزرار تعمل)

### 1. **📩 طلب خدمة جديدة**

#### الوظيفة:
تقديم طلب جديد للحصول على خدمة من الشركة

#### البيانات المطلوبة:
- **نوع الخدمة**: اختيار من قائمة الخدمات المتاحة
- **عنوان الخدمة**: عنوان واضح للطلب
- **الوصف**: شرح مفصل لما تحتاجه
- **الميزانية** (اختياري): المبلغ المتوقع للمشروع

#### النتيجة:
✅ إنشاء مشروع جديد في النظام
✅ إخطار فريق الشركة بالطلب الجديد
✅ تحديث إحصائيات لوحة التحكم
✅ رسالة تأكيد بنجاح الطلب

#### مثال على الاستخدام:
```
1. انقر على "طلب خدمة جديدة"
2. اختر: تطوير مواقع ويب
3. أدخل العنوان: موقع تجارة إلكترونية
4. وصف مفصل: موقع للبيع الإلكتروني برسوم ديناميكية
5. ميزانية: 100000 دينار
6. انقر "إرسال الطلب"
```

---

### 2. **💬 مراسلة الفريق**

#### الوظيفة:
التواصل مع فريق العمل بشأن مشروع معين

#### البيانات المطلوبة:
- **رقم المشروع**: معرف المشروع الذي تريد التواصل بشأنه
- **الرسالة**: نص الرسالة التفصيلية

#### النتيجة:
✅ تسجيل الرسالة في سجل المشروع
✅ إخطار الفريق المسؤول عن المشروع
✅ محفوظة في السجل الكامل للمشروع

#### مثال على الاستخدام:
```
1. انقر على "مراسلة الفريق"
2. أدخل رقم المشروع: 5
3. الرسالة: "حسب اتفاقنا، يجب إضافة ميزة الدفع"
4. انقر "إرسال الرسالة"
```

---

### 3. **📋 عرض الفواتير**

#### الوظيفة:
الاطلاع على جميع الفواتير والمدفوعات

#### البيانات المعروضة:
- **عنوان المشروع**: اسم المشروع
- **المبلغ**: قيمة الفاتورة بالدينار الجزائري
- **حالة الدفع**: مدفوع / معلق / قيد المراجعة
- **التاريخ**: تاريخ إصدار الفاتورة
- **نوع الخدمة**: نوع الخدمة المقدمة

#### حالات الدفع:
- 🟢 **مدفوع**: تم دفع الفاتورة بالكامل
- 🟡 **معلق**: الفاتورة في انتظار الدفع
- 🔵 **قيد المراجعة**: الفاتورة تحت المراجعة

#### مثال على الاستخدام:
```
1. انقر على "عرض الفواتير"
2. يتم عرض قائمة بجميع الفواتير
3. تحقق من حالة كل فاتورة
4. ادفع الفواتير المعلقة عند الحاجة
```

## 🐛 استكشاف الأخطاء

### المشكلة: الإحصائيات لا تظهر
**الحل:**
1. تحقق من اتصالك بالإنترنت
2. تأكد من تشغيل الخادم (يجب أن ترى "🚀 Server running")
3. افتح Browser Console (F12) وابحث عن الأخطاء
4. أعد تحميل الصفحة (F5)

### المشكلة: الأزرار لا تستجيب
**الحل:**
1. تحقق من أنك دخلت (قم بتسجيل الدخول أولاً)
2. تأكد من البيانات المدخلة صحيحة
3. ابحث عن رسائل الخطأ في Console (F12)
4. حاول مرة أخرى بعد ثوان

### المشكلة: لا يوجد بيانات في الفواتير
**الحل:**
1. تأكد من وجود مشاريع مكتملة
2. تحقق من حالة الدفع للمشاريع
3. انتظر 1-2 ثانية للتحميل
4. أعد تحميل الصفحة
