تحويل لوحة التحكم من صورة ثابتة إلى نظام حقيقي:

### قبل الآن:
```javascript
animateCounter(el, 3);      // رقم ثابت
animateCounter(el, 1);      // رقم ثابت
```

### الآن:
```javascript

const data = await response.json();
animateCounter(el, data.openRequests);        // من DB
animateCounter(el, data.activeProjects);      // من DB
```

## 📡 الـ API Endpoints

### جلب الإحصائيات
```bash
GET /api/projects/stats/dashboard
Header: Authorization: Bearer {token}
```

### إنشاء طلب
```bash
POST /api/dashboard/service-request
Header: Authorization: Bearer {token}
Body: { service_id, title, description, budget }
```

### إرسال رسالة
```bash
POST /api/dashboard/message
Header: Authorization: Bearer {token}
Body: { message, project_id }
```

### جلب الفواتير
```bash
GET /api/dashboard/invoices
Header: Authorization: Bearer {token}
```
