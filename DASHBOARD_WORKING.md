## 📡 API Endpoints

جميع Endpoints متاحة وتعمل:

### الإحصائيات
```
GET /api/projects/stats/dashboard
Header: Authorization: Bearer {token}
```

### الخدمات المتاحة
```
GET /api/dashboard/services
Header: Authorization: Bearer {token}
```

### الفواتير
```
GET /api/dashboard/invoices
Header: Authorization: Bearer {token}
```

### طلب خدمة جديد
```
POST /api/dashboard/service-request
Header: Authorization: Bearer {token}
Body: {
  "service_id": 1,
  "title": "عنوان الخدمة",
  "description": "وصف مفصل",
  "budget": 10000
}
```

### إرسال رسالة
```
POST /api/dashboard/message
Header: Authorization: Bearer {token}
Body: {
  "project_id": 1,
  "message": "نص الرسالة"
}
```

### تسجيل دخول
```
POST /api/auth/login
Body: {
  "email": "user@example.com",
  "password": "password"
}
```

## 🐛 استكشاف الأخطاء

### إذا لم تظهر الإحصائيات:
1. تحقق من اتصالك بالإنترنت
2. تأكد من تشغيل الخادم
3. افتح Browser Console (F12)
4. ابحث عن رسائل الخطأ

### إذا لم يستجيب الزر:
1. تحقق من أن لديك token (قم بتسجيل الدخول أولاً)
2. تأكد من البيانات المدخلة صحيحة
3. افتح Console وابحث عن الخطأ

### إذا لم تظهر الفواتير:
1. تأكد من وجود مشاريع مكتملة
2. تحقق من حالة الدفع
3. انتظر تحميل البيانات (1-2 ثانية)
