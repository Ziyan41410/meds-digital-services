4️⃣ 🚀 خطوات البدء السريع:
───────────────────────────────────────────────────────────────────────────────

⏱️ الوقت المقدر: 10 دقائق

STEP 1: التحضير
───────────────
1. تأكد من تثبيت:
   • Node.js v14+ (https://nodejs.org)
   • MySQL 5.7+ (https://www.mysql.com)
   • Git (https://git-scm.com)

2. افتح Terminal/Command Prompt

STEP 2: استنساخ المشروع
──────────────────────
cd your-folder
git clone https://github.com/yourname/meds-digital-services.git
cd meds-digital-services

STEP 3: تثبيت المكتبات
──────────────────────
npm install

سيتم تثبيت جميع المكتبات المطلوبة (قد يستغرق 2-3 دقائق)

STEP 4: إعداد قاعدة البيانات
───────────────────────────
1. افتح MySQL Command Line أو MySQL Workbench

2. قم بتشغيل:
   mysql -u root -p < database/meds_schema.sql

أو انسخ محتويات database/meds_schema.sql وشغله في Workbench

STEP 5: تعديل متغيرات البيئة
────────────────────────────
1. افتح الملف .env.local

2. عدّل القيم التالية:
   
   DB_HOST=localhost          # عنوان خادم MySQL
   DB_USER=root               # اسم مستخدم MySQL
   DB_PASSWORD=               # كلمة مرور MySQL
   
   JWT_SECRET=your_secret_key # مفتاح سري قوي
   
   SMTP_HOST=smtp.gmail.com
   SMTP_USER=your_email@gmail.com
   SMTP_PASS=your_password
   
   SERVER_PORT=3000           # المنفذ

STEP 6: تشغيل المشروع
──────────────────────
وضع التطوير (مع Nodemon):
npm run dev

أو الإنتاج:
npm start

STEP 7: فتح المتصفح
──────────────────
انتقل إلى:
http://localhost:3000

8️⃣ 🐛 استكشاف الأخطاء الشائعة:
───────────────────────────────────────────────────────────────────────────────

❌ مشكلة: "Cannot find module 'express'"
✅ الحل:
   npm install
   أو
   npm install express

❌ مشكلة: "Connection refused at localhost:3306"
✅ الحل:
   1. تأكد من تشغيل MySQL
   2. تحقق من البيانات في .env.local
   3. جرب: mysql -u root -p

❌ مشكلة: "Database 'meds_digital_services' doesn't exist"
✅ الحل:
   تشغيل ملف Schema:
   mysql -u root -p < database/meds_schema.sql

❌ مشكلة: "Port 3000 already in use"
✅ الحل:
   غير المنفذ في .env.local:
   SERVER_PORT=3001
   أو أغلق البرنامج الذي يستخدم 3000

❌ مشكلة: "CORS error" في المتصفح
✅ الحل:
   تحقق من allowedOrigins في app.js
   أضف origin جديد إن لزم الأمر

❌ مشكلة: "JWT Error: invalid token"
✅ الحل:
   جرب تسجيل الدخول مرة أخرى
   امسح localStorage من المتصفح

❌ مشكلة: "nodemon: command not found"
✅ الحل:
   npm install -g nodemon
   أو استخدم: npx nodemon backend/server.js
